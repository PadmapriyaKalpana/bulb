# 💡 Bulb Toggle — Light Switch App

A simple interactive web page that toggles a light bulb on and off using HTML, CSS, and JavaScript.

---

## 📁 Project Structure

```
project/
├── index.html        # Main HTML file
├── pic_bulbon.gif    # Image of a glowing bulb (ON state)
└── pic_bulboff.gif   # Image of a dark bulb (OFF state)
```

---

## 🚀 Getting Started

### Prerequisites

- A web browser (Chrome, Firefox, Edge, Safari)
- No installations or dependencies required

### Setup

1. **Clone or download** this project to your local machine.
2. **Download the bulb images** and place them in the same folder as `index.html`:
   - [pic_bulbon.gif](https://www.w3schools.com/js/pic_bulbon.gif)
   - [pic_bulboff.gif](https://www.w3schools.com/js/pic_bulboff.gif)
3. **Open `index.html`** in your browser.

---

## 🎮 How to Use

| Button | Action |
|--------|--------|
| **Turn on the Light** | Switches the bulb image to the ON (glowing) state |
| **Turn off the Light** | Switches the bulb image to the OFF (dark) state |

---

## 🧠 How It Works

The page uses two JavaScript functions to swap the `src` attribute of an `<img>` element:
Each button calls one of these functions via an `onclick` attribute, updating the displayed image instantly.
